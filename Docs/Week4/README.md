## 📊 Risk Register Framework

The project risk register tracks eleven critical points of exposure, deeply analyzing two core rows per category alongside three high-impact overlapping system vulnerabilities:

*   **AI-Technical:** Model drift over time; subgroup underperformance (specifically monitored among elderly patient cohorts).
*   **Operational:** Workflow abandonment; staff adoption failure due to clinical friction.
*   **Ethical:** Clinician overreliance / automation bias; accountability gaps in machine-assisted decision making.
*   **Equity:** Differential algorithmic performance based on patient age and primary language status.
